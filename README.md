# Plateforme cartographique de visualisation de projets au Cameroun

## Creating a project

```bash
# clone this repo
git clone git@github.com:Tonhauck/plateforme-urbaine-cameroun.git

# install dependencies
cd plateforme-urbaine-cameroun
npm i 
# or using pnmp
pnpm i

# run the local serve
npm run dev
```
